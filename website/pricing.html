Frequently Asked Questions

Is there a free version available?

Yes, the Local Only version is completely free with no time limits. It includes full functionality using local Ollama models.

What does the one-time payment include?

The $19 one-time payment unlocks API access to cloud models (OpenAI, Claude, etc.) and includes all premium features. There are no recurring fees.

Can I upgrade from the free version later?

Yes, you can purchase API access at any time with a one-time $19 payment. Your data and settings will be preserved.

What payment methods do you accept?

We accept all major credit cards and PayPal for the one-time API access purchase. Enterprise plans also support bank transfers.

Can I use Cindy offline?

Yes, both plans include offline mode with local models. The API Access plan includes enhanced models for better offline performance.

How does the knowledge vault work?

The knowledge vault automatically saves and indexes your research. You can search your personal vault to retrieve information from past research sessions. API Access provides 15GB storage vs 5GB in Local Only.
