Installation Instructions

macOS Installation

  1. Download the appropriate .dmg file for your Mac
  2. Open the downloaded file and drag Cindy to Applications
  3. Launch Cindy from Applications folder
  4. Follow the setup wizard to get started

Windows Installation

  1. Download the .exe installer
  2. Run the installer as administrator
  3. Follow the installation prompts
  4. Launch Cindy from the Start menu

Linux Installation

  1. Download the appropriate package for your distribution
  2. Install using your package manager (dpkg for .deb, rpm for .rpm)
  3. Launch from applications menu or terminal
  4. Complete the initial setup
